HTML

The components of html elements are:

  1. Start Tag
  2. Attributes
  3. Content
  4. End tag

Void elements are elements when they don't have an End Tag like <br> or <hr>! They are denoted with a slash (/) before the closing angle bracket (>) in the same start tag. It is optionally to do the slash before the closing angle bracket!

Attributes provide additional information about elements and are always specified in the start tag! They consist of a name and a value, separated by an equals sign (=)!

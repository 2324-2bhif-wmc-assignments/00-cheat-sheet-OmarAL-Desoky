Lists and Tables

Lists

Tags:

  • bullet is square (list-style-type: square;)
  • bullet is image (list-style-image: url("my-bullet.png");)
  • bullet uppercase roman number (list-style-type: upper-roman;)
  • bullet position to be inside (list-style-position: inside;)
  • bullet position to be outside (list-style-position: outside;)

Tables

Tags:

  • Black dotted border (border: 1px dotted black;)
  • Solid border of 1 px width having a red color and rounded corners(border: 1px solid red; border-radius: 5px;)
  • Merge the borders of table cells so that they appear as one line (border-collapse: collapse;)

Box Model

Syntax

  • Margin: margin: 1rem;The space between the border and the surrounding elements is 1rem.
  • Padding padding: 1rem;The space between the border and the content is 1rem.
  • Border border: 1rem black solid;The border is black solid and is 1 rem! it can also be any color or dashed!

Basic

General

The components of html elements are:

  1. Start Tag
  2. Attributes
  3. Content
  4. End tag

Void elements are elements when they don't have an End Tag like <br> or <hr>! They are denoted with a slash (/) before the closing angle bracket (>) in the same start tag. It is optionally to do the slash before the closing angle bracket!

Attributes provide additional information about elements and are always specified in the start tag! They consist of a name and a value, separated by an equals sign (=)!

Classes and IDs

Classes

Classes are used to select html elements and can occur more than once on a page! The HTML class attribute is used to specify a class for an HTML element.
Start Tag of an element of p of class message: <p class="message">

IDs

IDs are used to select html elements and can NOT occur more than once on a page! The HTML id attribute is used to specify a unique id for an HTML element.
Start Tag of an element of article of id main: <article id="main">

Box Model

General

Explanation of the different parts:
The box model allows us to add a border around elements, and to define space between elements.

  • Content - The content of the box, where text and images appear!
  • Padding - Padding is the space between the content of the element and its border. Clears an area around the content.
  • Border - A border that goes around the padding and content!
  • Margin - Margin is the space between the border and the surrounding elements! Clears an area outside the border.

Display elements

Block Element

A block-level element always starts on a new line and takes up the full width available (stretches out to the left and right as far as it can)! Can have margin and padding to keep distance from all surrounding elements! Has already a little distance between all other elements! (use an extra line above and below)

Examples of block-level elements:

  • <div>
  • <p>
  • <h1 - h6>

Inline Element

An inline element does not start on a new line and only takes up as much width as necessary. Stay in the line with their neighboring elements! Only keep distance only to the elements on the left and on the right if a positive margin and/or padding is defined (only right/left)

Examples of inline elements:

  • <span>
  • <a>
  • <img>
